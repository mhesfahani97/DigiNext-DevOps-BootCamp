# Dockerized WordPress Deployment 🚀

## Project Objective 🎯
Our mission is clear and concise:
1. Launch a WordPress blog with ease using Docker Compose.
2. Ensure the longevity of your data by implementing persistence, safeguarding it against container restarts.

# Let's Dive into Action 🛠️
Let's embark on this journey into the world of Docker Compose and WordPress, paving the path to success! 🌐🏹

Here's what you need to do:

1. Open your terminal and navigate to the project directory.
2. Run the following command to deploy the WordPress application and its dependencies in the background:

```bash
docker-compose up -d
```

3. After the command completes, open your web browser and enter the following URL to access your WordPress site:

```
http://localhost:8085
```
